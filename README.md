# Word 文档排版工具

基于 Python 的 Word 文档批量排版工具，支持 `.doc` / `.docx` 双格式，兼容 Microsoft Office 和 WPS。

## 功能

- **页面设置** — 上下左右页边距（mm）
- **正文样式** — 中英文字体、字号、颜色、加粗/斜体
- **段落设置** — 行距（倍行距/固定值/最小值）、首行缩进、左右缩进、段前段后间距、对齐方式
- **标题样式** — 支持标题 1~6 级独立设置（字体、字号、颜色、对齐、缩进、行距、段前段后）
- **批量处理** — 选择文件或文件夹，一键处理
- **格式预设** — 内置三种预设：公文标准格式、学术论文格式、简约商务报告
- **`.doc` 兼容** — 通过 `win32com` 自动将 `.doc` 转为 `.docx` 后排版
- **安全输出** — 原文件不做修改，输出文件保存为 `原文件名-Revise.docx`

## 安装

```bash
pip install python-docx pywin32
```

> 仅处理 `.docx` 时无需 `pywin32`；需处理 `.doc` 文件则必须安装。

## 使用

```bash
python Word_Editor.py
```

1. 导入文件（单个 / 文件夹）
2. 在左侧和中栏调整排版参数，或从 `format_presets.json` 中选择预设
3. 点击 **开始排版**

## 预设格式

| 预设 | 正文字体 | 字号 | 行距 | 适用场景 |
|---|---|---|---|---|
| 公文标准格式 | 仿宋 | 三号(16pt) | 固定28pt | 政府公文、红头文件 |
| 学术论文格式 | 宋体 | 小四(12pt) | 1.5倍行距 | 毕业论文、期刊投稿 |
| 简约商务报告 | 微软雅黑 | 五号(10.5pt) | 1.15倍行距 | 商业报告、企划书 |

## 技术栈

- `python-docx` — .docx 读写
- `win32com` — .doc 格式转换（调用 Word/WPS COM）
- `tkinter` — GUI 界面，三栏布局
